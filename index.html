Text Comparison Tool

Created by John Mark Esplana | GitHub Repository

Instructions

  1. Paste or type your original text into the left textarea.
  2. Paste or type your modified text into the right textarea.
  3. Click the "Compare" button to see the differences.
  4. Review the results:
    • Added content is highlighted in green.
    • Removed content is highlighted in red.
    • Word count, character count, and number of additions/removals are shown at the top of each column.
